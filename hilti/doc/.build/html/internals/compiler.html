Checker¶

hilti.checker.checkAST(ast)¶

Verifies the correctness of the AST ast, as returned for example by hilti.parser.parse(), and returns the number of errors found. If any errors are detected, suitable error messages are written to standard error.

This method is the primary way of ensuring that a HILTI AST is well-formed, and should be called whenever an AST could violate any semantic constraints being relied upon by the code generator. In particular, ASTs should be checked after parsing and canonification.
