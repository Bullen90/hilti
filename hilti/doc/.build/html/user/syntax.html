Instruction Reference¶

In the following, we list the complete set of instructions that the HILTI language provides. For reference, we give them with both their textual mnemonics (as one uses when writing a HILTI program in a text editor), along with their Python class names as provided by the HILTI Python module.

Control Flow¶

class hilti.instructions.flow.Call(op1, op2, target)¶
<none or any> = call <function> <tuple>

Calls function using the tuple in op2 as arguments. The argument types as well as the target’s type must match the function’s signature.

class hilti.instructions.flow.CallC(op1, op2, target)¶
<none or any> = call.c <function> <tuple>

For internal use only.

Calls the external C function using the tuple in op2 as arguments. The argument types as well as the target’s type must match the function’s signature.

class hilti.instructions.flow.CallTailResult(op1, op2, op3, target)¶
<none or any> = call.tail.result <function> <tuple> <label>

Like Call(), calls function using the tuple in op2 as arguments. The argument types must match the function’s signature. The function must return a result value and it’s type must likewise match the function’s signature. Different than Call(), CallTailVoid must be the last instruction of a Block. After the called function returns, control is passed to block op3.

This instruction is for internal use only. During the code generation, all Call instructions are converted into either CallTailVoid or CallTailResult instructions.

class hilti.instructions.flow.CallTailVoid(op1, op2, op3)¶
call.tail.void <function> <tuple> <label>

For internal use only.

Calls function using the tuple in op2 as arguments. The argument types must match the function’s signature, and the function must not return a value. Different than Call(), CallTailVoid must be the last instruction of a Block. After the called function returns, control is passed to block op3.

This instruction is for internal use only. During the code generation, all Call instructions are converted into either CallTailVoid or CallTailResult instructions.

class hilti.instructions.flow.IfElse(op1, op2, op3)¶
if.else <bool> <label> <label>

Transfers control label op2 if op1 is true, and to op3 otherwise.

class hilti.instructions.flow.Jump(op1)¶
jump <label>

Jumps unconditionally to label op2.

class hilti.instructions.flow.ReturnResult(op1)¶
return.result <none or int>

Returns from the current function with the given value.

The type of the value must match the function’s signature.

class hilti.instructions.flow.ReturnVoid¶
return.void

Returns from the current function without returning any value.

The function’s signature must not specifiy a return value.

Type Integer¶

The +integer+ data type represents signed integers of a fixed width. The width

Parameters

  • 16+ for a 16 is specified as part of the type name as, e.g., in +int: bit integer.
class hilti.instructions.integer.Add(op1, op2, target)¶
<int> = int.add <int> <int>

Calculates the sum of the two operands. Operands and target must be of same width. The result is calculated modulo 2^{width}.

class hilti.instructions.integer.Div(op1, op2, target)¶
<int> = int.div <int> <int>

Divides op1 by op2, flooring the result. Operands and target must be of same width. If the product overflows the range of the integer type, the result in undefined.

Throws DivisionByZero if op2 is zero.

class hilti.instructions.integer.Eq(op1, op2, target)¶
<bool> = int.eq <int> <int>

Returns true iff op1 equals op2.

class hilti.instructions.integer.Ext(op1, target)¶
<int> = int.ext <int>

Zero-extends op1 into an integer of the same width as the target. The width of op1 must be smaller or equal that of the target.

class hilti.instructions.integer.Lt(op1, op2, target)¶
<bool> = int.lt <int> <int>

Returns true iff op1 is less than op2.

class hilti.instructions.integer.Mul(op1, op2, target)¶
<int> = int.mul <int> <int>

Multiplies op1 with op2. Operands and target must be of same width. The result is calculated modulo 2^{width}.

class hilti.instructions.integer.Sub(op1, op2, target)¶
<int> = int.sub <int> <int>

Subtracts op1 one from op2. Operands and target must be of same width. The result is calculated modulo 2^{width}.

class hilti.instructions.integer.Trunc(op1, target)¶
<int> = int.trunc <int>

Bit-truncates op1 into an integer of the same width as the target. The width of op1 must be larger or equal that of the target.
